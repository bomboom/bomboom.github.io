MVC

Model-View-Controller(MVC) is a software architectural pattern for implementing user interfaces, in order to separate the presentation of information which is accepted from users or presented to users.

Develop Environment

JDK 1.8.0, Tomcat 7.0, MySQL 5.5, Windows 7, Eclipse

Main Packages

  • JavaBean: maintain classes related to MySQL relations, such as users, orders, products.
  • JavaServlet: maintain servlet maps to realize the data transmission from back-end to front-end.
  • JavaDao: maintain functions to supply the data according to user actions, such as search products, browsing order history.
  • Tools: maintain connection to MySQL.

Pass Data from one jsp page to Another

When we open a web page, we can find there is always a symbol “?” in URL. That’s a mark of dynamic pages. If item.id = 1, Then the url would be productdetail.jsp?name=1. When you want get name in another page, you need only use request.getParameter(“name”).

Servlet Actions

To make a dynamic page, we need to define actions in jsp and through web xml control to execute the right servlet map in back-end.
When we define a action whatever we use form or input in html, we should then define a servlet map in web.xml.
